The Ovorost Application Programming Interface (API) let developpers interact programmatically with Ovorost . The API offers two response types: XML (default output format) and JSON. Each request take a 'format' parameter to specify the output.
Available as soon as possible.
GET /api/version
Retreive Ovorost version.
GET /api/news
Get Ovorost news.
GET /api/user-infos
Get informations about an user.
Parameter :
  • email of the user.
POST /api/user-infos
Update informations about an user.
Parameters :
  • email of the user.
  • password
  • first-name
  • last-name
  • longitude
  • latitude
  • sex
GET /api/event
Get user's ski events.
Parameter :
  • email : email of the user.
GET /api/resorts
Get informations about a ski resorts.
Parameter :
  • name : the resort's name.
